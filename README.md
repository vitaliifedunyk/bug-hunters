# vanilla-template

A reusable Vite starter template for vanilla HTML, CSS, and JavaScript projects with partial-based page structure and a ready-to-edit starter layout.

## Overview

This repository is intended as a clean starting point for small frontend projects without frameworks. It includes a Vite setup, shared styles, HTML partial support, and a neutral starter page that can be replaced quickly when beginning a new build.

## Included in the Template

- Vite development and build workflow;
- `src/`-based project structure;
- HTML partial support for reusable layout sections;
- base CSS setup with reset, container, and starter layout styles;
- lightweight starter markup for faster project bootstrapping;
- GitHub Pages deployment workflow.

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Vite

## How to Use

1. Clone the repository:

```bash
git clone https://github.com/vitaliifedunyk/vanilla-template.git
```

2. Open the project folder.
3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Replace the starter content in `src/` with your project-specific markup, styles, and scripts.

## Notes

- The build configuration is prepared for local development and GitHub Pages deployment.
- The starter content is intentionally minimal and meant to be replaced.
- The repository name is used automatically for the production base path in CI.
