# Меблерія — Furniture Store Catalog

Проєкт команди розробників **Bug Hunters**. Це сучасний фронтенд-додаток для онлайн-каталогу меблевого магазину, розроблений з акцентом на зручність інтерфейсу, повну адаптивність (Mobile-First) та продуктивність.

## 📋 Опис проєкту

Вебсайт "Меблерія" надає користувачам можливість:
- Переглядати каталог якісних меблів.
- Ознайомлюватись із детальною інформацією про кожен товар (галерея зображень, описи, доступні кольори, габарити).
- Залишати заявки на замовлення та зворотний зв'язок через форми з надійною валідацією.
- Переглядати додаткову інформацію (наприклад, FAQ чи умови) за допомогою зручних акордеонів.

## 🛠 Використані технології та бібліотеки

Проєкт побудовано на основі ванільного стеку з використанням сучасних інструментів:

- **HTML5 & CSS3** (Методологія BEM, Flexbox, Grid)
- **JavaScript (ES6+)**
- **Vite** — збирач проєкту та локальний dev-сервер.
- **Swiper** — реалізація каруселей та слайдерів товарів.
- **Accordion-js** — для роботи з розгортанням контенту (FAQ).
- **iziToast** — для відображення спливаючих повідомлень (нотифікацій) користувачу після сабміту форм.
- **Axios** — для роботи з HTTP запитами.
- **vite-plugin-html-inject** — для модульного розбиття розмітки (HTML partials).

## 🚀 Ключовий функціонал (Features)

- **Strict Mobile-First адаптивність**: Інтерфейс плавно масштабується від смартфонів (320px-375px) до широких екранів (1440px+).
- **Складні модальні вікна**: Кастомні модалки товарів та форм замовлення з чітко розрахованими відступами та галереями.
- **Custom UI елементи**: 
  - Динамічний вибір кольору товару.
  - Блок рейтингу (css-star-rating).
  - Валідація інпутів у формах (з візуальними підказками про помилки).

## 💻 Інструкція із запуску

Для розгортання проєкту локально виконайте наступні кроки:

1. Схилюйте репозиторій:
   ```bash
   git clone https://github.com/vitaliifedunyk/bug-hunters.git
   ```
2. Встановіть необхідні NPM-пакети:
   ```bash
   npm install
   ```
3. Запустіть локальний сервер для розробки:
   ```bash
   npm run dev
   ```
   Проєкт автоматично відкриється у браузері за посиланням `http://localhost:5173/`.

Для збірки фінальної (production) версії:
```bash
npm run build
```
